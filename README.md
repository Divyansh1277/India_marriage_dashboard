# India Marriage Insights Dashboard

## 📊 Overview

This project presents an interactive dashboard for analyzing marriage-related data in India. It explores relationships between various socio-economic and cultural factors such as:

* Education level
* Income level
* Inter-caste and inter-religion marriages
* Dowry practices
* Parental approval
* Marital satisfaction
* Divorce trends

The dashboard is built using **Panel** and **hvPlot**, allowing users to interactively explore the dataset.

---

## 🚀 Features

* Interactive visualizations
* Clean data preprocessing and transformation
* Dynamic filtering and exploration
* Insightful analysis of marriage patterns

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/india-marriage-dashboard.git
cd india-marriage-dashboard
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the dashboard using:

```bash
panel serve india_marriage_dashboard_updated.ipynb --show
```
